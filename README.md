# Mobile-Device-Management-System
Project Title: Mobile Device Management (MDM) System

Area of Interest: Full Stack / Backend Development

Project Details:
Developed a Mobile Device Management system to centrally track devices, manage application versions, and control update rollouts. Built the backend using Spring Boot with REST APIs and developed the frontend using React, Vite, JavaScript, and Tailwind CSS. The system supports device registration, version lifecycle management, scheduled updates, downgrade prevention, and audit logging to ensure secure and controlled deployment of app updates across devices.
